# [Uniswap/protocol-fees] feat(v4): replace pair fees with per-family pairClassFees

> 출처: GITHUB | 2026-07-08 | 관련 항목: [[FM-4]], [[FM-6]]

## 요약
Uniswap v4의 페어별 수수료 구조를 'pairClassFees'로 대체하여 제품군별 수수료를 개별 설정합니다.
애그리게이터와 네이티브 수수료 설정의 결합을 분리하여 독립적인 수수료 배포를 지원합니다.

## 시그널
v4 수수료 시스템에서 애그리게이터와 프로토콜 자체 수수료 설정을 분리하여, 보다 정밀하고 유연한 멀티체인 및 외부 연동 수수료 제어가 가능해집니다.
